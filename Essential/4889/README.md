## 문제 요약
- `{`와 `}`만으로 이루어진 문자열이 주어질때, 안정적인 문자열을 만들기 위한 최소 연산 수 구하기
- 연산은 2가지
  - `{`를 `}`로 바꾸기
  - `}`를 `{`로 바꾸기

## 문제 풀이
```java
for (int i = 0; i < str.length(); i++) {				
  if (str.charAt(i) == '{') stack.push('{');
  else {
    if (stack.isEmpty()) {
                  answer++;
                  stack.push('{');
              } else {
                  stack.pop();
              }
  }
}
```
- 문자열의 각 문자를 탐색한다
  - 현재 문자가 `{`이면 push
  - 현재 문자가 `}`이면
    - 비어있는데 닫는 괄호이면 여는 괄호로 바꿔줘야하니까 answer++ 하고 `{`를 push
    - 비어있지 않으면 스택에 들어있는 `{`와 한 쌍을 이루므로 pop

```java
answer += stack.size() / 2;
```
- 모든 문자를 다 돌았는데 스택에 남아있는 `{`들은 짝이 없는 것들. -> 그중 절반을 `}`로 바꾸면 안정적이게 된다
