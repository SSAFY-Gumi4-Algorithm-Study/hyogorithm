## 문제 요약

- `n`개의 문제의 데드라인과 컵라면 수가 주어진다
- 데드라인 안에 문제를 풀면 컵라면 수만큼 받을 수 있다
- 문제를 푸는데는 단위시간 1이 걸린다
- 받을 수 있는 최대 컵라면 수 구하기

## 문제 풀이

```java
Arrays.sort(problems, (a, b) -> a[0] - b[0]);
```

- 문제를 데드라인 기준으로 정렬
  - `problems[i][0]` : 데드라인
  - `problems[i][1]` : 컵라면 수

```java
for (int i = 0; i < n; i++) {
    pq.add(problems[i][1]);
    if (pq.size() > problems[i][0]) pq.poll();
}
```

- 문제의 컵라면 수를 하나씩 `pq`에 저장
    - 현재 문제의 데드라인보다 지금까지 선택한 문제 수가 더 많으면 컵라면수가 가장 적은 문제를 제거
    - 그러면 기한 내에 받을 수 있는 보상이 큰 문제들만 `pq`에 남는다

```java
while (!pq.isEmpty()) {
    answer += pq.poll();
}
```

- `pq`에 남은 값들을 모두 더한다
